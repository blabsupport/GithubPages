[バックアップ] urBackup を使ってみる
🐶　バックアップソフトの選定候補として sota Backup, BunBackup 以外にも選択肢がほしい。
😃　できれば世代管理あり、エラースキップに対応しているソフトがいいですよね。
BunBackup の増分バックアップが、思いのほか時間がかかってしまう。結局、ファイル個体ごとに対象かどうかチェックが入るから仕方がない。とはいえ、時間を短縮したいシナリオは突発で振ってくるので無視できない。

🐶　ArcServe には有償版、トライアル版、無償版があるみたいな。今回は Windows Server 2022 IOT が対象なのだが、無償版はクライアント OS 限定みたいだし。
🐶　ChatGpt に聞いてみよう。

以下は無料で使用できるサーバーOS対応のバックアップソフトです。

対応OS
Windows Server
Linux
特徴
ファイルとディスクイメージのバックアップが可能で、簡単に使えるWebベースのインターフェースを提供しています。
物理サーバーとクライアントOSの両方をサポートしており、差分バックアップや増分バックアップも可能です。
制限 GUIやサポートにおいて有償版ほどの高度な機能はありませんが、基本的なバックアップには十分。 
C:\Windows\system32>fsutil 8dot3name query g:
  ボリュームの状態は 0 です (8dot3 名の作成は有効です)。
  レジストリの状態は 2 です (ボリューム単位で設定します - 既定値)。
  
  以上の設定に基づいて、8dot3 名の作成は g: で有効です

  C:\Windows\system32>fsutil 8dot3name set g: 1
  8dot3name の生成が g: で無効になりました

  C:\Windows\system32>fsutil 8dot3name query g:
  ボリュームの状態は 1 です (8dot3 名の作成は無効です)。
  レジストリの状態は 2 です (ボリューム単位で設定します - 既定値)。
  
  以上の設定に基づいて、8dot3 名の作成は g: で無効です
